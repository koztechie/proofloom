# ProofLoom

An AI-verified public skill-building tracker with a live leaderboard. Built for the H0 Hackathon.

## MVP Scope

**IN SCOPE:**
- Auth (register/login with Credentials)
- Create skill challenge (target 7-365 days)
- Daily proof submission form (text + optional URL)
- Amazon Bedrock Nova Lite proof evaluation (score 0–100 + comment)
- Public profile page `/u/[handle]` with contribution heatmap
- Global leaderboard `/leaderboard` with category filter
- Vercel Cron streak expiry check
- Static `/pricing` page (demonstrating business model, no billing)

**OUT OF SCOPE:**
- Real payments / Stripe integration
- OAuth (GitHub/Google login)
- Email notifications
- File/image uploads
